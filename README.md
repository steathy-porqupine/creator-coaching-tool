# OnlyFans Creator Coaching Tool

A comprehensive coaching tool for content creators to master persuasive, non-pushy engagement and level up their income.

## Features

- **Practice Scenarios**: Interactive practice exercises with feedback on messaging approaches
- **Quick Scripts**: Copy-paste ready scripts for off-platform and on-platform messaging
- **Weekly Playbook**: Strategic messaging calendar and templates
- **Fan Segmentation**: Tier-based strategy for different subscriber types
- **Track Results**: Key metrics to track and A/B testing guide
- **Red Flags**: Common mistakes to avoid and win formula

## Getting Started

1. Open `index.html` in your web browser
2. Navigate between tabs using the navigation buttons
3. Practice your messaging in the Practice Scenarios tab
4. Copy ready-to-use scripts from the Quick Scripts tab
5. Follow the Weekly Playbook for consistent engagement

## Features Breakdown

### Practice Scenarios
- 5 real-world scenarios with pushy vs. persuasive examples
- Interactive text areas to practice your responses
- Instant feedback on your messaging approach

### Quick Scripts
- Off-platform scripts (Twitter, TikTok, Instagram DMs)
- On-platform scripts (OnlyFans DMs)
- One-click copy functionality

### Weekly Playbook
- 4-day messaging calendar (Monday, Wednesday, Friday, Sunday)
- Sample messages for each day
- Pro tips for mass messaging

### Fan Segmentation
- Tier 1: Active Buyers (5-15% of subs)
- Tier 2: Engaged Non-Payers (20-35% of subs)
- Tier 3: Lurkers (50-75% of subs)
- Identification guide and tailored strategies

### Track Results
- 5 key metrics with target percentages
- A/B testing quick guide
- Weekly tracker template

### Red Flags
- 8 common mistakes to avoid
- Fix strategies for each mistake
- The Win Formula (5 key principles)
